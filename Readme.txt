SBER TEST TASK
Динамическая таблица с информацией о сотрудниках. Можно добавлять новых сотрудников, указывать их дату рождения и зарплату. Сохраняется в базе данных.
При отрисовке таблицы на бэке высчитывается кол-во полных лет и налог с зарплаты для каждого сотрудника. Можно удалять ячейку и сотрудника.
Стек: 
Node Js, Express, MongoDB, Mongoose, React, Bootstrap
